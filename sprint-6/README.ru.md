### Functionality: 
* 10 initial cards on page load
* Popup for adding cards
* Like function for every card
* Delete function for every card
* Additionally:
  * Fixed like button size
    
### Project tech stack:
1. HTML 5
2. CSS 3
3. JavaScript (ES6)
